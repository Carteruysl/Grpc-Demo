# Grpc-Demo
Grpc-Demo

簡易說明
實作一個可以透過 grpc 的方式進行資料交換的伺服器。
範例的內容為德州撲克的規則，依照輸入的撲克牌類型，回覆對應的牌型。

運行方式

1.go run server.go

2.使用 BloomRPC，先匯入 proto 檔，就可以運行測試。
